# Requisitos

-  Python 3.10 ou superior
-  Numpy 2.4.1 ou superior

# Execução

- python3 main.py

# Relativos aos testes

As instâncias de testes usadas estão no diretório "tests/instancies", enquanto os resultados dos testes na pasta "tests/results". Todos os resultados dos testes estão no arquivo "resultsTable.txt".

Da maneira como o seu conteúdo foi formatado, ele pode ser copiado e colado facilmente para uma planilha de excel. Basta selecionar todo o seu conteúdo ("Ctrl+A") e o copiá-lo ("Ctrl+C") e, por último, colá-lo.

Os dados relativos às estruturas de dados de "sStar" de cada testes pode ser encontrada nas passtas "tests/results/U0.02" e "tests/results/U0.05"
